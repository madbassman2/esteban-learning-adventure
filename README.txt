ESTEBAN'S LEARNING ADVENTURE

WHAT'S INCLUDED
- 5 themed worlds
- 25 levels
- Receptive identification activities
- Visual scanning activities
- Memory matching games
- Spoken prompts and praise
- Star scoring and saved progress
- Balloon reward game
- Offline-ready PWA support
- Works on Windows, Mac, Android, iPhone, and iPad

TEST ON WINDOWS
1. Extract the ZIP.
2. Double-click index.html.
3. For full offline/install behavior, open Command Prompt in the extracted folder and run:
   python -m http.server 8000
4. Open http://localhost:8000 in Chrome or Edge.

ANDROID
1. Host the folder using GitHub Pages, Netlify, or another static web host.
2. Open the URL in Chrome.
3. Choose Add to Home screen / Install app.

IPHONE OR IPAD
1. Open the hosted URL in Safari.
2. Tap Share.
3. Tap Add to Home Screen.

NOTES
- Progress saves only on the device/browser being used.
- No accounts, ads, tracking, or internet connection are required after installation.
